mplementation Plan: Dynamic Multi-Pair Orderbook (MT5 to Web)Dokumen ini merincikan langkah teknis membangun aplikasi Web Orderbook yang secara otomatis membaca daftar pair dari Market Watch MetaTrader 5.1. Arsitektur SistemData Source: MetaTrader 5 Terminal (Market Watch).Backend: Python (Auto-discovery symbols).Frontend: HTML/JS (Auto-generate Tabs).Protocol: WebSocket (JSON Data).2. Fitur Utama (Upgrade)Dynamic Symbol Fetching: Tidak perlu tulis nama pair di kodingan. Cukup tambah pair di MT5 Market Watch, web otomatis update.Smart Suffix Handling: Otomatis membuang akhiran "r" atau "z" di tampilan web agar bersih.Scrollable UI: Menu tab bisa digeser jika pair yang dimasukkan banyak.3. Langkah Implementasi (Step-by-Step)Phase 1: Backend Logic (bridge.py)Script Python harus lebih pintar dari sebelumnya:Symbol Discovery:Gunakan fungsi mt5.symbols_get(visible=True).Ini akan mengambil semua simbol yang aktif di jendela Market Watch.Suffix Logic (Auto-Detect):Iterasi setiap simbol.Simpan real_symbol (contoh: XAUUSDr) untuk request data.Buat display_symbol (contoh: XAUUSD) dengan membuang huruf terakhir jika itu suffix akun Raw.WebSocket Protocol (Handshake):Saat klien connect, kirim pesan awal: TYPE: SYMBOL_LIST. Isinya daftar nama pair untuk membuat Tab.Setelah itu, loop mengirim pesan: TYPE: MARKET_DATA untuk update grafik.Phase 2: Frontend Logic (index.html)HTML tidak boleh punya tombol mati (hardcoded).UI Layout:.tab-container harus kosong saat awal load.CSS .tab-container harus overflow-x: auto (bisa discroll ke samping) dan white-space: nowrap.JavaScript Logic:Listen event WebSocket.Jika terima TYPE: SYMBOL_LIST -> Loop array-nya -> Create Element <button> -> Masukkan ke .tab-container.Jika terima TYPE: MARKET_DATA -> Cek apakah simbol cocok dengan tab aktif -> Render Canvas.Interaction:Klik Tab -> Ganti variabel currentSymbol -> Request repaint canvas.4. Cara Penggunaan (User Manual)Setup di MT5:Buka MT5 Desktop.Tekan Ctrl + M (Market Watch).Klik kanan -> Symbols.Masukkan semua pair yang diinginkan (Gold, Forex, Crypto, Stocks).Pastikan pair tersebut muncul di list Market Watch.Jalankan Server:Run python bridge.py.Server akan mencetak log: "Found 15 symbols in Market Watch".Buka Web:Buka index.html.Menu di atas akan otomatis muncul sesuai list di MT5.5.
