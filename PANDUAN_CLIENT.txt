üî• FITUR BARU: BOOKMAP / HEATMAP (FUTURES/CYBER MODE)
------------------------------------------------------------------------

Di Dashboard Utama, klik tombol "OPEN PRO BOOKMAP" di pojok kanan atas.
Ini akan membuka jendela baru khusus untuk melihat "Whale Walls".

Fitur Bookmap:
- TOMBOL XAUUSD / BTCUSD: Pindah pair dengan cepat di pojok kanan atas.
- GARIS PUTIH: Adalah harga saat ini.
- BARIS WARNA-WARNI: Adalah tumpukan order (Limit Orders).
  > MERAH = Order Jual (Resistance)
  > HIJAU = Order Beli (Support)
- KOTAK TEBAL: Menandakan "Big Player" atau Whale sedang pasang posisi.

------------------------------------------------------------------------
‚ùì TROUBLESHOOTING (SOLUSI MASALAH)
------------------------------------------------------------------------

A. "Tidak ada data / Chart diam saja"
   - Cek internet Anda.
   - Pastikan jendela hitam (Console) masih terbuka. Jangan ditutup!
   - Tekan F5 di browser untuk refresh.

B. "Tombol XAUUSD/BTCUSD tidak bisa diklik"
   - Sudah diperbaiki di update terakhir.
   - Coba refresh browser (Ctrl + R).

C. "Muncul error ''Python not found''"
   - Anda perlu install Python. Download di python.org (versi 3.12).
   - Saat install, CENTANG opsi "Add Python to PATH".
